🩺 MedPredict

An AI-powered, multi-disease prediction system built to empower early health decisions with machine learning.

About the Project

MedPredict is a machine learning-powered web platform that helps users assess their health risks for Heart Disease, Diabetes, Kidney Disease, and Liver Disease using basic medical parameters. By combining data-driven insights with a clean and secure interface, the system enables early detection and timely medical action — all without requiring any database or paid service.

Key Features

7+ ML Algorithms

Uses Logistic Regression, Decision Tree, Random Forest, KNN, SVM, Gradient Boosting, AdaBoost & Stacked Ensemble.

Personal Dashboard

Track all your past predictions with charts and history sorted by disease and date.

Hospital Locator

Find real hospitals near your GPS location using free OpenStreetMap data.

PDF Reports

Download a professional medical-style prediction report to share with your doctor.

No Database Needed

All data is stored locally in JSON files. No MySQL, no cloud storage, no external APIs.

100% Free

No subscriptions, no API keys, no paid services. Completely open and free to run locally.
